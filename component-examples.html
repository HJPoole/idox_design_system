29. Empty State

Copy only markup between ai-example-start and ai-example-end markers.

Use when a list, table, or panel has no content to display. Centre an icon, a short heading, a supporting sentence, and an optional call-to-action. The icon should use text-tertiary and a large size to create visual breathing room without competing with surrounding content.
